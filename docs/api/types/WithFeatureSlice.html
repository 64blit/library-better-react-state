    Type Alias WithFeatureSlice<TFeatures, K>

    WithFeatureSlice: {
        [P in K as `${string & K}Slice`]: FeatureSlice<TFeatures[K]>
    }

    Represents the props expected by a component that requires access to a specific feature slice. This utility type is typically used with Higher-Order Components (HOCs) or context patterns to inject a feature slice into a component's props.

    The resulting prop name will be ${featureName}Slice (e.g., if K is "user", the prop will be userSlice).

    Type Parameters

    • TFeatures extends FeatureDefinitions

      The overall feature definitions for the store, as passed to createFeatureStore.

    • K extends keyof TFeatures

      The key (name) of the specific feature slice required by the component.

    Example

    type MyComponentProps = OwnProps & WithFeatureSlice<AppFeatures, 'user'>;
    // MyComponentProps will now require a prop named `userSlice` of type `FeatureSlice<AppFeatures['user']>`.
